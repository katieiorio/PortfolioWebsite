Javascript Fitness Calendar

Javascript

HTML

CSS

For my first JavaScript endeavor, I decided to challenge myself by creating a fitness calendar inspired by my go-to workout application. I began this project with HTML and CSS to craft the calendar's appearance. My aim was to maintain a minimal, user-friendly design, so I opted for a conventional table layout. To ensure adaptability across all devices, I added responsive features, employing media queries for seamless mobile integration.

With the visual aesthetics complete, I moved on to making the calendar interactive using JavaScript. Users can log their daily exercises by inputting the activity type, date, and time. Upon submission, an illustrative icon appears within the corresponding calendar box, depicting the user's activity. Leveraging event listeners, functions, conditional statements, and DOM manipulation, the calendar is fully functional.

Each phase of this project was a learning experience, enriching my understanding of JavaScript, HTML, and CSS. While there's always room for growth, I'm pleased to say that I achieved the envisioned objective of this personal project.
